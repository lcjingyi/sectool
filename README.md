#### 一个简单命令使用工具

可编译使用使用`main.exe -h`查看帮助
```
Small tools

Usage:
  myApp [command]

Available Commands:
  base            base编码
  completion      Generate the autocompletion script for the specified shell
  deduplicateFile 文件去重功能
  dirScan         dir扫描
  fileEncrypto    文件加解密
  help            Help about any command
  ping            ping
  portScanf       端口扫描
  subdomain       子域名爆破
  url             url编码

Flags:
  -h, --help   help for myApp

Use "myApp [command] --help" for more information about a command.
```
